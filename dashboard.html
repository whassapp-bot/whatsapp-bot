📊 Panel de Gestión de Proyectos

📋 ¿Cómo usar este panel?

✏️ 1. Actualiza la información: Cambia el estado, progreso, hitos y acciones de cada proyecto aquí.

💾 2. Haz clic en "Guardar Cambios": Los cambios se guardan automáticamente en el servidor.

📧 3. Reportes automáticos: Cada día a las 09:00 hs y 16:00 hs recibirás un email con toda esta información actualizada.

➕ 4. Agregar acciones: Usa el campo de texto para agregar nuevas acciones pendientes.

💡 Este panel se conecta directamente con tu bot. Cuando guardas cambios, el próximo reporte incluirá toda la información actualizada.
